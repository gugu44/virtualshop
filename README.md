# 가상 피팅 스튜디오 (Virtual Shop)

React + TypeScript로 만든 간단한 드래그 앤 드롭 착장 스튜디오 프로토타입입니다. 더미 JSON으로 쇼핑몰 상품을 불러와 모델 영역 위에 배치해 코디를 미리 구성할 수 있습니다.

## 빠른 시작

> 네트워크가 차단된 환경이라 `npm install`이 동작하지 않을 수 있습니다. 필요한 패키지는 `package.json`에 정의되어 있으니 네트워크가 허용된 환경에서 설치 후 실행하세요.

```bash
npm install
npm run dev
```

## 기능
- 더미 데이터(상의/하의/아우터/ACC) 드래그 앤 드롭 지원
- 모델 영역에 아이템을 여러 개 배치하거나 위치 이동
- 카테고리 필터, 리셋 버튼 제공
